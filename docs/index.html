О проекте

Проект «Туристическое бюро» представляет собой консольное приложение на языке C++, предназначенное для автоматизации учета экскурсионных маршрутов, автобусов, экипажей и выполненных рейсов.

Приложение поддерживает аутентификацию пользователей, работу с базой данных SQLite, выполнение CRUD-операций и формирование отчетов по автобусам, экипажам и маршрутам.

Документация проекта размещена в первом репозитории в GitHub Wiki, а исходный код и сборка проекта находятся во втором публичном репозитории.

Основные возможности

  • Учет экскурсионных маршрутов
  • Учет автобусов и закрепленных за ними экипажей
  • Хранение сведений о выполненных рейсах
  • Расчет доходов экипажей за выбранный период
  • Выполнение SQL-запросов и формирование отчетов
  • Разграничение доступа пользователей
Проект разрабатывается в рамках лабораторной работы с использованием UML, SQLite, GitHub Wiki, GitHub Pages и GitHub Actions.
