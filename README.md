# Проект: Туристическое бюро

## Описание проекта

Система предназначена для автоматизации работы туристического бюро, осуществляющего обслуживание экскурсионных маршрутов.

Бюро организует перевозку туристов с использованием автобусов и экипажей, закреплённых за каждым транспортным средством. Стоимость обслуживания зависит от параметров маршрута и количества участников экскурсии.

Система позволяет хранить и обрабатывать информацию о маршрутах, автобусах, экипажах и выполненных рейсах.

---

## Предметная область

Система оперирует следующими сущностями:

- **Маршруты** (название, начальный и конечный пункт, протяжённость)
- **Автобусы** (номер, название, пробег)
- **Члены экипажа** (ФИО, табельный номер, стаж, категория, адрес, год рождения)
- **Рейсы** (даты, маршрут, количество пассажиров, стоимость билета)
