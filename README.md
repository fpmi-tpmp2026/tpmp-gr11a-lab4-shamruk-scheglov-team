# Project Name

**Туристическое бюро (Tourist Bureau)** — консольное приложение для автоматизации работы туристического бюро.

---

# Description

Система предназначена для автоматизации работы туристического бюро, осуществляющего обслуживание экскурсионных маршрутов.  
Приложение позволяет хранить и обрабатывать данные о маршрутах, автобусах, экипажах и рейсах.  
Пользователь может управлять данными (добавление, редактирование, удаление) и формировать отчёты.  
Проект реализован на языке C++, использует SQLite для хранения данных, CMake для сборки и GoogleTest для тестирования.

---

# Installation

## Требования

- macOS
- CMake 3.14 или выше
- Компилятор C++ (clang или gcc)
- SQLite3
- Git

## Установка зависимостей на macOS

```bash
brew install cmake sqlite3
xcode-select --install
```

## Клонирование и сборка проекта

```bash
git clone https://github.com/Polina707-lab/tpmp-lab4-work.git
cd tpmp-lab4-work

mkdir build
cd build
cmake ..
cmake --build .
```

## Инициализация базы данных

```bash
cd ..
sqlite3 data/tourist_bureau.db < sql/init.sql
```

## Сборка и запуск тестов

```bash
cd build
ctest --output-on-failure
```

# Usage

## Запуск приложения

```bash
./bin/app
```

## Использование

После запуска пользователь может:

- пройти аутентификацию;
- управлять маршрутами, автобусами и рейсами;
- добавлять, редактировать и удалять данные;
- формировать отчёты;
- просматривать результаты в текстовом формате.
