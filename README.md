  # Project Name

**Tourist Bureau** — консольное приложение для автоматизации работы туристического бюро.

# Description

Tourist Bureau — это приложение для учёта экскурсионных маршрутов, автобусов, экипажей и выполненных рейсов.  
Система позволяет выполнять аутентификацию пользователей и работать с данными в базе SQLite.  
Пользователь может добавлять, изменять и удалять маршруты, автобусы и рейсы, а также получать отчёты по работе бюро.  
Проект реализован на языке C++, собирается с помощью CMake и содержит unit-тесты на базе GoogleTest.

# Installation

## Требования

- macOS или Linux
- CMake 3.14+
- Компилятор C++ (clang или gcc)
- SQLite3
- GoogleTest
- Git

## Установка зависимостей на macOS

```bash
brew install cmake sqlite3 googletest
xcode-select --install
```

## Клонирование репозитория

```bash
git clone https://github.com/Polina707-lab/tpmp-lab4-work.git
cd tpmp-lab4-work
```

## Сборка проекта

```bash
cmake -S . -B build
cmake --build build
```

## Запуск тестов

```bash
ctest --test-dir build --output-on-failure
```

# Usage

## Запуск приложения

## Возможности системы

### 1. Функционал администратора

Администратор имеет полный доступ к системе:

#### Управление рейсами
| Операция | Описание |
|----------|----------|
| Просмотр рейсов | Вывод рейсов по автобусу за выбранный период |
| Добавление рейса | Создание нового рейса |
| Обновление рейса | Изменение данных рейса |
| Удаление рейса | Удаление рейса из системы |

#### Управление автобусами
| Операция | Описание |
|----------|----------|
| Просмотр автобусов | Список всех автобусов |
| Добавление автобуса | Регистрация нового автобуса |
| Обновление автобуса | Изменение данных автобуса |
| Удаление автобуса | Удаление автобуса |

#### Управление маршрутами
| Операция | Описание |
|----------|----------|
| Просмотр маршрутов | Список всех маршрутов |
| Добавление маршрута | Создание нового маршрута |
| Обновление маршрута | Изменение маршрута |
| Удаление маршрута | Удаление маршрута |

#### Отчёты и аналитика
| Отчёт | Описание |
|------|----------|
| Статистика по автобусу | Данные по рейсам, пассажирам и доходу |
| Доход экипажей | Начисления экипажам за период |
| Самый дорогой маршрут | Информация о максимальной стоимости |
| Автобус с максимальным пробегом | Статистика по пробегу |
| Начисления экипажу | Выплаты на выбранную дату |

#### Работа с изображениями
| Операция | Описание |
|----------|----------|
| Загрузка изображения | Сохранение изображения автобуса в БД |
| Выгрузка изображения | Сохранение изображения в файл |
| Информация об изображении | Проверка наличия и размера |
